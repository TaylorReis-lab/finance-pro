<h2>Overview</h2>
<p>Finance-pro is a powerful React-based platform tailored for financial management and analytics. It offers a modular architecture combined with a comprehensive set of UI components, enabling developers to craft intuitive, data-driven dashboards with ease. The core features include:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0">🎨 <strong>Color Mode Support:</strong> Seamless switching between light and dark themes for enhanced user experience.</li>
<li class="my-0">📊 <strong>Data Visualization:</strong> Rich charts, grids, and analytics components for insightful data presentation.</li>
<li class="my-0">🧩 <strong>Reusable Components:</strong> Custom icons, menu buttons, stat cards, and date pickers designed for flexibility and consistency.</li>
<li class="my-0">🚀 <strong>Robust Routing &amp; Navigation:</strong> Intuitive routing structure and responsive navigation components for smooth user journeys.</li>
<li class="my-0">🛠️ <strong>Developer-Centric Design:</strong> Well-organized source code, centralized theming, and extensive customization options to streamline development and maintenance.</li>
</ul>

<h2>Getting Started</h2>
<h3>Prerequisites</h3>
<p>This project requires the following dependencies:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0"><strong>Programming Language:</strong> TypeScript</li>
<li class="my-0"><strong>Package Manager:</strong> Npm, Yarn</li>
</ul>
<h3>Installation</h3>
<p>Build finance-pro from the source and install dependencies:</p>
<ol>
<li class="my-0">
<p><strong>Clone the repository:</strong></p>
<pre><code class="language-sh">❯ git clone https://github.com/TaylorReis-lab/finance-pro
</code></pre>
</li>
<li class="my-0">
<p><strong>Navigate to the project directory:</strong></p>
<pre><code class="language-sh">❯ cd finance-pro
</code></pre>
</li>
<li class="my-0">
<p><strong>Install the dependencies:</strong></p>
</li>
</ol>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">❯ npm install
</code></pre>
<p><strong>Using <a href="https://yarnpkg.com/">yarn</a>:</strong></p>
<pre><code class="language-sh">❯ yarn install
</code></pre>
<h3>Usage</h3>
<p>Run the project with:</p>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">npm start
</code></pre>
<p><strong>Using <a href="https://yarnpkg.com/">yarn</a>:</strong></p>
<pre><code class="language-sh">yarn start
</code></pre>
<h3>Testing</h3>
<p>Finance-pro uses the {<strong>test_framework</strong>} test framework. Run the test suite with:</p>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">npm test
</code></pre>
<p><strong>Using <a href="https://yarnpkg.com/">yarn</a>:</strong></p>
<pre><code class="language-sh">yarn test
</code></pre>
